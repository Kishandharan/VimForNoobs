Installation

First of all, we first need Vim Installed in our machine to do anything in Vim.
So if you are on linux, use the command line to install Vim with your package manager.
If you are on macOS, first install the brew package manager and use the command line to install Vim.
And to open Vim, type in vim in your command line to open it.
Alternatively, you can use vim as a extenstion in a code editor/IDE too.
Like the Vim extension in VS code.
