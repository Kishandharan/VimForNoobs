The Normal Mode

In the previous lession, we discussed about the different modes.
So in this lession, we will dicuss about the normal mode.
But before diving in, let's see what is the Normal Mode.


What is Normal Mode?
Normal mode is the most important mode in vim.
It is the mode we need to be in to switch to other modes.
When we first open up vim, this is the mode we are in by default.

Command: dd
This command is used for deleting the line our cursor is currently at!
For example, let's say we had a file with the following content ("<c>" indicates our cursor):
"
I 
dummsadfsadt<c>
Love
Mechanical
Keyboards
".
Now the line our cursor is at has to be removed to form a meaningful sentence.
For that we can just do "dd" to delete that whole line instead of smashing the backspace key!
