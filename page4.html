The Normal Mode

In the previous lession, we discussed about the different modes.
So in this lession, we will dicuss about the normal mode.
But before diving in, let's see what is the Normal Mode.


What is Normal Mode?
Normal mode is the most important mode in vim.
It is the mode we need to be in to switch to other modes.
When we first open up vim, this is the mode we are in by default.

Command: dd
This command is used for deleting the line our cursor is currently at!
For example, let's say we had a file with the following content ("<c>" indicates our cursor):
"
I 
dummsadfsadt<c>
Love
Mechanical
Keyboards
".
Now the line our cursor is at has to be removed to form a meaningful sentence.
For that we can just do "dd" to delete that whole line instead of smashing the backspace key!


Command: u
This command is used to undo something that you have done! It is like Ctrl+z on windows. 
For example lets say we had a file with the following content, text between "<u>" and "</u>" indicates the change to be undone:
"
Some random dude wrote down this file.
So we have to definitely <u>delete</u> this file
".
Now we can undo the word "delete" by simply pressing u instead of manually deleting it.


Command: yy
This command is used to copy the line our cursor is at.
For example lets say we was in line four in this file:
"
basdfkhsf
aslidfkljsdf
klsjdfkljasdf
some random content
sdlfjljsdf
".
Now to copy this whole line we can just do yy.


Command: D
This command is very similar to the dd command, but the key difference is that dd deletes the whole current line no matter where we are in the line.
But D deletes only till the end of the line, from where our cursor is in the current line.
For example in the line "this is my songzozoz", if our cursor was in the first occurrence of the letter z,
we can do D to delete the text "zozoz" to form a meaningful sentence.


Command: S
This command is very similar to the dd command, 
except S enters the insert mode after deletding the line while dd doesn't.
